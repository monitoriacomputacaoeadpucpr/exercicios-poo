# Exercício 02

Imprima a tabuada de multiplicação até 10 de um número inteiro digitado pelo usuário no terminal.

### Exemplo:

Digite um número inteiro: `2`

Resultado:
```
2 x 0 = 0
2 x 1 = 2
2 x 2 = 4
2 x 3 = 6
2 x 4 = 8
2 x 5 = 10
2 x 6 = 12
2 x 7 = 14
2 x 8 = 18
2 x 9 = 18
2 x 10 = 20
```

### Conteúdos:
- Entrada de dados
- Saída de dados
- Variáveis e tipos de dados
- Expressões aritméticas
- Concatenação de Strings
- Loop for
